### 2026년 YouTube Shorts 알고리즘 핵심 (Shortimize 기반)

[피드 노출] → [Stayed to Watch: 스와이프 vs 시청] → [Engaged Views: 초반 이후 잔존] → [APV/AVD: 평균 시청 비율/시간] → [만족도 신호: 좋아요/싫어요/설문]


**핵심 메트릭:**

- **Stayed to Watch** = 훅 성능 (스크롤 멈춤 비율)
- **APV (Average Percentage Viewed)** = 스토리 + 페이싱 성능
- **Engaged Views** = 2025.03.31 이후 도입, YPP 수익 기준 (루프 미포함)
- **벤치마크**: APV 70-90%, 참여율 5-10%, 30초 리텐션 과반수 유지

### 바이럴 숏츠의 구조적 공식 (Virvid 118K 영상 분석)

HOOK (0-3초) → 결과/충격 먼저 보여주기, 패턴 인터럽트 PROBLEM (3-15초) → 공감 유발, 고통점 제시 (75단어 = 30초) SOLUTION (15-45초) → 가치 전달, 2-4초마다 점프컷 CTA (마지막 5초) → 구체적 행동 유도 ("A댓글 or B댓글")


### 훅 유형 10가지 (검증된 1M+ 뷰)

| # | 유형 | 예시 | 심리 기제 |
|---|---|---|---|
| 1 | Mystery Hook | "이걸 알고 있었나요...?" | 호기심 갭 |
| 2 | List Hook | "충격적인 사실 TOP 3" | 완료 충동 |
| 3 | Problem Hook | "당신이 하는 3가지 실수" | 자기 동일시 |
| 4 | Shock Hook | "이건 믿을 수 없지만..." | 고각성 감정 |
| 5 | Call-out Hook | "무협 팬이라면 멈춰!" | 직접 지목 |
| 6 | Transformation | "X에서 Y로 변한 비결" | 열망 자극 |
| 7 | Exclusivity | "1%만 아는 비밀" | 희소성 |
| 8 | Controversial | "지금까지 알던 게 전부 틀렸다" | 인지 충돌 |
| 9 | FOMO | "이거 들어봤어?" | 사회적 배제 공포 |
| 10 | Urgency | "읽기 전에 이것부터 봐라" | 긴급성 |

### 리텐션 유지 기법 (JoinBrands + Shortimize)

- **1-2초마다 시각 변화**: 줌 레벨, 앵글, 오버레이, 텍스트 변경
- **마이크로 오픈 루프**: 5-8초마다 "하지만 진짜 충격은..."

- **자막 필수**: 무음 시청자 85% — 볼드, 고대비, 키네틱 타이포그래피
- **루프 구조**: 마지막 프레임 → 첫 프레임 자연 연결, 리워치 유도
- **6학년 수준 어휘**: 즉각적 이해 = 2배 조회수

## 🔴 현재 로직 vs 최정상 기법 GAP 분석

### 현재 우리 숏츠 제작 로직

롱폼 스크립트에서 200자 잘라옴 → 정적 이미지 1-5장 (16:9) → Ken Burns 줌/팬 (단일 효과) → ASS 자막 (고정 위치, 단일 스타일) → TTS 나레이션 → FFmpeg 단순 조립


### 최정상 숏츠 로직 (리서치 기반 목표)

숏츠 전용 스크립트 (HOOK→PROBLEM→SOLUTION→CTA) → 9:16 세로 이미지 (인물 클로즈업) → 다층 시각 효과 (줌+팬+쉐이크+플래시+분할) → 키네틱 자막 (단어별 하이라이트, 크기 변화, 등장 애니메이션) → 다중 오디오 레이어 (TTS + BGM + SFX) → 루프 연결 (끝→시작 자연 전환) → 훅 텍스트 오버레이 (첫 1.5초 전용)

| **이미지 구도** | 와이드 풍경샷 | 인물 클로즈업, 감정 표현, 세로 구도 | 프롬프트에 구도 지시 |
| **시각 효과** | Ken Burns 5종 (줌/팬 단일) | 줌+팬+쉐이크+플래시+회전, 2-4초마다 전환 | 효과 엔진 대폭 확장 |

| **자막 타이밍** | 전체 나레이션 균등 분배 | 음성 싱크 (단어 단위 타임스탬프) | edge-tts SSML 활용 |

| **첫 프레임 디자인** | 나레이션 시작과 동일 | 결과/충격 이미지 + 대형 텍스트, 무음 이해 가능 | 훅 프레임 전용 생성 |
| **페이싱** | 이미지당 균등 시간 | 씬별 긴장도에 따른 가변 시간, 빠른 구간 존재 | 동적 타이밍 배분 |
| **CTA** | 없음 | 마지막 2-3초 구독/댓글 유도 텍스트 + 음성 | CTA 자동 삽입 |

### 키네틱 자막 (drawtext 애니메이션)
```bash
# 단어별 팝업 등장 (아래→위 슬라이드)
drawtext=text='충격':fontsize='min(60,60*(t-0.5)/0.2)':
x=(w-text_w)/2:y='h-200+max(0,30*(1-(t-0.5)/0.2))':
fontcolor=white:borderw=3:bordercolor=black:
enable='between(t,0.5,2.0)'

# 핵심 단어 색상 하이라이트
drawtext=text='고수':fontcolor=yellow:fontsize=70:enable='between(t,1.0,2.5)'
임팩트 쉐이크 효과
Copy# 프레임 오프셋으로 흔들림
crop=iw:ih:x='if(between(t,2.0,2.5),random(0)*20-10,0)':y='if(between(t,2.0,2.5),random(1)*20-10,0)'
화이트 플래시 전환
Copy# 씬 전환 시 0.15초 화이트 플래시
fade=t=out:st=4.85:d=0.15,fade=t=in:st=5:d=0.15
훅 텍스트 오버레이 (첫 1.5초)
Copy# 반투명 배경 + 대형 텍스트
drawbox=x=0:y=h*0.35:w=iw:h=h*0.3:color=black@0.6:t=fill:enable='between(t,0,1.5)',
drawtext=text='숨겨진 고수의 운명은?':fontsize=72:fontcolor=white:
  x=(w-text_w)/2:y=(h-text_h)/2:borderw=4:bordercolor=black:
  enable='between(t,0,1.5)'
